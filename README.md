# recommender_systems
3 ways with recommenders

This notebook was created during my Data Science course at WBS Coding School (Berlin, Germany) in 2021

The notebook shows 3 different styles of recommending movies to customers:
1. Simply the top rated movies in our database (excluding those with very few rating - popularity plays a small role)
2. "As you watched ..., we recommend ..." (movie recommendation based upon another movie that you've watched)
3. Personal recommendations (trying to predict what a user would like - using the movie reviews of other users, and the similarities of other users)

This was purely an exploratory project to see the underlying workings of recommenders. It's not intended to be a production recommender.
